JOURNAL – QUICK START

1. Plug in the USB.

2. Open:
Mood Tracker.exe
(This launches the journal automatically.)

3. Lock Screen

Type your password

Click Unlock
⚠ If you forget your password, the journal cannot be recovered.

4. Connect your data file

Click Connect data file (top right)

Choose: mood-entries.json (inside the app folder)

5. Write your entry

Pick the date

Type your journal entry

Click Save

6. Next time

Run Mood Tracker.exe again

Enter the same password

Click Connect data file
(Choose the same mood-entries.json)

7. Changing your password

Unlock normally

Make sure the data file is connected

Click Change password

Follow the steps
Use this new password from now on.

8. Important
Do not delete mood-entries.json.
It contains your encrypted journal.
